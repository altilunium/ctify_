### How to install
1. Copy everything to the htdocs folder.

<img width="1271" height="614" alt="image" src="https://github.com/user-attachments/assets/ad165cfd-9c24-43c9-b40f-41c701cba599" />


2. Setup the master password. Go to local/config.php , modify the pmcrypt(’yourpasswordhere’).

<img width="604" height="101" alt="image" src="https://github.com/user-attachments/assets/0e800a1f-4708-4d32-9a47-609ac2a38348" />

3. Continue the [initial setup task](http://cyllchuesnconii.rf.gd/PmWiki/InitialSetupTasks).
